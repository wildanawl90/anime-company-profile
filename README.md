## 🚀 Cara Menggunakan

### Buka Langsung
Buka file `index.html` di browser favorit Anda.

### Local Server
```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve .

# PHP
php -S localhost:8000
```

Kemudian buka `http://localhost:8000`

## 🎨 Kustomisasi

### Ganti Warna
Edit file `style.css` - bagian `:root`:

```css
:root {
    --primary: #ff6b9d;      /* Pink utama */
    --secondary: #c44569;     /* Pink gelap */
    --accent: #f8b500;        /* Emas */
    --neon-blue: #00d9ff;     /* Biru neon */
}
```

### Ganti Konten
Edit `index.html`:
- Ganti teks di setiap section
- Ganti gambar dari Unsplash
- Update informasi tim
- Ubah kontak info

##3 Ganti Gambar
Ganti URL gambar di `index.html` dengan URL gambar Anda:
```html
<img src="URL_GAMBAR_ANDA" alt="Deskripsi">
```
